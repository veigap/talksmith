# Frases a eliminar

## Aperturas de carraspeo

Frases que anuncian en vez de decir. Borralas y afirmá el contenido directo.

- "La verdad es que"
- "Lo cierto es que"
- "Seamos honestos:"
- "Hablemos claro:"
- "Esta es la cuestión:"
- "El punto es el siguiente:"
- "Resulta que"
- "La realidad es que"
- "No es ningún secreto que"
- "Todos sabemos que"
- "Déjame decirte algo:"
- "Te lo digo sin vueltas:"
- "Hay algo que nadie dice:"
- "La verdad incómoda es"
- "Lo que pocos entienden es"

Cualquier construcción "esto es lo que..." / "aquí está el problema..." es carraspeo antes del punto. Cortala y decí el punto.

## Muletillas de énfasis

No agregan significado. Borralas.

- "Punto." / "Así de simple." / "Así de claro."
- "Dejá que eso te cale." / "Pensalo un segundo."
- "Esto importa porque"
- "No te confundas" / "Que no te engañen"
- "Y esto es clave:"
- "Ojo con esto:"

## Jerga corporativa y de LinkedIn

Reemplazá por lenguaje llano.

| Evitar | Usar en cambio |
|--------|----------------|
| Navegar (desafíos, incertidumbre) | Manejar, resolver |
| Desglosar / desmenuzar (un tema) | Explicar, examinar |
| Abordar (una problemática) | Tratar, resolver, encarar |
| Apalancar / poner en valor | Aprovechar, usar |
| Sinergia | Trabajo conjunto (o nombrá qué hacen juntos) |
| Hoja de ruta | Plan |
| Un antes y un después / game-changer | Importante (o mostrá el cambio concreto) |
| Redoblar la apuesta | Insistir, invertir más |
| Análisis en profundidad / deep dive | Análisis |
| Dar un paso atrás para ver el panorama | Reconsiderar |
| De cara al futuro / hacia adelante | Después, desde ahora |
| Retomar el tema más adelante | Volver a esto |
| Estar alineados / en la misma página | De acuerdo |
| Potenciar | Mejorar, aumentar (o el verbo concreto) |
| Ecosistema (de startups, de contenidos) | Sector, ambiente, mercado |
| Panorama / paisaje (del marketing, etc.) | Situación, campo |

## Adverbios en -mente

Matá los adverbios. Ni suavizadores, ni intensificadores, ni coberturas.

Reincidentes específicos:

- "realmente"
- "literalmente"
- "genuinamente"
- "honestamente"
- "sinceramente"
- "simplemente"
- "básicamente"
- "profundamente"
- "verdaderamente"
- "fundamentalmente"
- "inherentemente"
- "inevitablemente"
- "curiosamente"
- "interesantemente"
- "sumamente"
- "increíblemente"
- "absolutamente"
- "totalmente"
- "claramente"
- "obviamente"
- "efectivamente"
- "precisamente"
- "justamente"

También caen los cuasi-adverbios de relleno: "de hecho", "en verdad", "sin duda", "por supuesto", "desde luego", "a decir verdad".

## Frases de relleno

- "En esencia" / "En el fondo" / "A fin de cuentas"
- "Al final del día"
- "En el mundo actual" / "En la actualidad" / "Hoy en día" / "Hoy por hoy"
- "En un mundo donde" / "En la era de" / "En tiempos de"
- "En el vertiginoso mundo de" / "En el cambiante panorama de"
- "A la hora de" (→ "al", "cuando")
- "Es importante mencionar que" / "Es importante destacar que"
- "Cabe destacar que" / "Cabe señalar que" / "Cabe mencionar que"
- "Vale la pena señalar que" / "No está de más recordar que"
- "En definitiva" / "En resumidas cuentas" / "En última instancia"
- "Dicho esto" / "Dicho lo anterior"
- "En este sentido" / "En esta línea"
- "Como bien sabemos" / "Como es sabido"

## Meta-comentario

Sacá los apartes autorreferenciales. El texto tiene que avanzar, no anunciar su propia estructura.

- "A lo largo de este artículo veremos..."
- "En esta sección exploraremos..."
- "Como veremos más adelante..."
- "Te voy a contar paso a paso..."
- "Acompañame en este recorrido..."
- "Pero eso da para otro post"
- "Spoiler:" / "Plot twist:" / "Atención:"
- "Ya lo sabés, pero"
- "Disfrazado de"
- "No es un bug, es un feature" (y sus variantes)

## Énfasis performativo

Intimidad falsa o sinceridad fabricada:

- "te lo prometo"
- "existen, te lo juro"
- "confía en mí"
- "y no exagero"
- "se cuela" / "se filtra" (como drama de algo abstracto que "se cuela" en tu vida)

## Anunciar en vez de mostrar

Declarar la dificultad o la importancia en lugar de demostrarla:

- "Esto es genuinamente difícil"
- "Esto es lo que el liderazgo significa en realidad"
- "Así se ve X de verdad"
- "lo que de verdad importa"
- "juega un papel fundamental" / "juega un rol clave"
- "es clave" / "es crucial" / "resulta esencial" (sin decir por qué ni para qué)

## Fórmulas de revelación

La plantilla "el verdadero X viene cuando / está en / aparece cuando" promete profundidad sin entregarla. Prohibida en todas sus variantes.

- "El verdadero cambio viene cuando"
- "El verdadero valor viene cuando"
- "El verdadero aprendizaje está en"
- "La verdadera pregunta es"
- "El verdadero desafío aparece cuando"
- "La verdadera magia sucede cuando"
- "Ahí es donde ocurre la magia"
- "Ahí está el verdadero X"

**En cambio:** Mostrá el X concreto. "El equipo mejoró cuando dejó de estimar en horas" le gana a "el verdadero cambio viene cuando abrazás la incertidumbre".

## Declarativas vagas

Oraciones que anuncian importancia sin nombrar la cosa concreta. Matálas.

- "Las razones son estructurales"
- "Las implicancias son profundas"
- "Este es el problema de fondo"
- "Hay mucho en juego"
- "Las consecuencias son reales"
- "El impacto es significativo"
- "Esto lo cambia todo"

Si una oración dice que algo es importante/profundo/estructural sin mostrar la cosa concreta, cortala o reemplazala por la cosa concreta.

## Léxico de IA en español

Palabras que casi ningún humano usa en ese contexto pero los modelos aman:

| Marca de IA | Alternativa humana |
|-------------|-------------------|
| "Sumérgete en" / "sumergirse en" | Empezá, mirá, leé |
| "Descubre" (imperativo de marketing) | Nombrá lo que hay |
| "Explora" (ídem) | Ídem |
| "Fascinante" / "apasionante" | Mostrá por qué interesa |
| "Crucial" / "vital" / "esencial" | Importante para X porque Y |
| "Robusto" (fuera de ingeniería) | Sólido, firme |
| "Amplio abanico" / "amplia gama" | Variedad (o enumerá dos) |
| "Sin fisuras" / "de manera fluida" | Decí qué pasa sin fricción |
| "El auge de" | X creció, más gente usa X |
| "Cabe preguntarse" | Preguntá o afirmá |
| "No es casualidad que" | Afirmá la causa |
| "Marcar la diferencia" | Decí qué cambia |
| "Un viaje" (de aprendizaje, del cliente) | Proceso, o nombrá las etapas |
| "Punto de partida" / "primer paso hacia" / "el comienzo del camino" | Decí el estado real: incompleto, versión inicial, borrador |
| "Tapiz" / "mosaico" (de culturas, ideas) | Mezcla, variedad |
| "Cabe recalcar" | Borrá y decí la cosa |
