# Presenter Profile

This file defines the **global presenter profile** that applies to every presentation produced in this repository. It is read once at the start of every session (between Step 0 and Step 1) and used as default context for thesis framing, audience defaults, slide tone, and class structure.

Fill in the sections below. Leave a section blank if it genuinely does not apply — the Presenter Agent treats blank sections as "ask me each time".

---

## How my presentations are consumed

Primarily **live talks** (in-person or remote). Slides function as punctuation; the speaker notes carry the talk.

## Audience defaults

**Technical peers** — engineers and researchers. Deep technical level, comfortable with jargon, want substance over scaffolding. Default to assuming familiarity with core concepts; don't over-define basics.
